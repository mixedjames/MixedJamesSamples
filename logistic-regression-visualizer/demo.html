Logistic Regression Visualizer

I found logistic regression tought to understand and most of the explanations started with descriptions of why the logistic function combined with linear regression gave a probability function. None of this seemed very intuitive to me so I made this simulator to help.

Some key ideas

Logistic regression is a binary classifier
That means it tries to decide if something is true or false based on it's inputs. That's all it is
It does that by finding a line (or a plane) that separates the trues from the falses.
